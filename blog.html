HTML y CSS [Parte 1]

Hola mi nombre es Dayan Betancourt, soy desarrolladora web hace 9 años, me pueden buscar en redes sociales como @delfinbeta. Este artículo es para compartir los primeros pasos en el mundo del desarrollo web con HTML y CSS.

¿Qué es el Desarrollo Web?

Es la creación de todo lo que ves en Internet, pero se resume en algo tan básico como la creación de un documento de texto que se compartirá en toda la web. El Desarrollo Web se puede dividir en múltiples especialidades como: Frontend, Backend, Base de Datos, Servidores, entre otras.

El Desarrollo Frontend consiste justamente en la creación de los documentos que se quieren compartir en la web y dicha creación se hace con HTML y CSS.

¿Qué es HTML?

HyperText Markup Language, es el lenguaje que entienden todos los navegadores web; no es un lenguaje de programación, es un lenguaje de marcado de hipertexto y representa el contenido de nuestro documento, pero más importante aún representa su estructura.

Hipertexto se refiere a texto con enlaces, que te permiten ir de un documento a otro y un lenguaje de marcado lo que permite es que el contenido del documento no sea solo texto, sino que es un texto con una estructura y dicha estructura se obtiene por un conjunto de etiquetas, esas etiquetas son las que conforman la sintaxis del lenguaje de marcado.

¿Qué es CSS?

Cascading Style Sheets, es otro lenguaje que entienden los navegadores y que le indica cómo será la presentación de la estructura definida con el HTML, se refiere a los estilos que se usarán para que nuestro documento resalte.

Mediante CSS se le indica al navegador el conjunto de reglas que definen, colores, fuentes, espaciados y de manera general la presentación visual que tendrá el documento.

HTML y su Estructura

La estructura es la que define cómo estará elaborado nuestro documento y la semántica que tendrá, todo esto se logra a través de etiquetas, que definen las diferentes secciones del documento. HTML tiene la estructura de un árbol, donde se van creando etiquetas con distintas jerarquías.

Para comenzar a escribir HTML sólo se necesita un editor de texto (ATOM, Brackets o Sublime Text) y un navegador web (Google Chrome, Firefox, Edge o Safari). Las etiquetas principales al escribir un documento HTML son:

<!DOCTYPE html> Indica el tipo de documento que estoy escribiendo.

<html lang="es"> </html> Indican el inicio y fin del documento; adicionalmente con el atributo lang se indica el idioma.

<head> </head> Indican las cabeceras del documento.

<body> </body> Indica el contenido a mostrar del documento.

 

Identación: las etiquetas que sean hijas de otra deben ir con un espacio de separación, definido por un tab o por 2 o 4 espacios en blanco, a gusto del programador.

El head: encierra la información de configuración que no se muestra como las etiquetas de metadatos. Por ejemplo:

<meta charset="utf-8"> Es importante para soportar los acentos y ñ.

Comentarios: son mensajes que van a aparecer en el código para ayudarnos a documentar o especificar información importante a recordar, pero que no aparecerán en la web. Para crear comentarios se utilizan:

<!-- Comentario -->
/* Comentario */
// Comentario

Tu Primera Web

Ahora escribe tu primera web, abre tu editor de texto incluso puede ser el bloc de notas y copia lo siguiente:

<!DOCTYPE html>
<html lang="es">
<head>
<meta charset="utf-8">
<title>Mi Primera Web</title>
</head>
<body>
<h1>Mi Primera Web</h1>
<p>Hola, yo soy @DelfinBeta y estoy creando mi primera web con un poco de texto.</p>
</body>
</html>

Guarda el archivo con el nombre de tu preferencia y la extensión .html y luego ábrelo en un navegador web.
